Request Format

The public api is accessed through making a HTTPS (port 443) OR HTTP (port 80) request to blazing-sails.nitbamiapp.com. Similar to most API's, the API is divided into several endpoints that each return their own responses (and take in their own set of parameters). The format for each request is as follows: https>//blazing-sails.bitnamiapp.com/php_rest_blazingsails/api/post/<endpoint>.php?api_key=&parameter=value...

Authentication

Authentication is done through the usage of a API KEY within the URL Parameters that get passed when making any web request.
